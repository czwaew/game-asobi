# JA業務ポータル

GitHub Pages向けの静的PWAです。

## ログイン
- 職員ID: `staff`
- パスワード: `1234`
- 管理者画面の初期化を試す場合は「管理者としてログイン」にチェック

## 公開方法
リポジトリ直下に、このフォルダ内の7ファイルをアップロードしてください。

## 注意
- データはブラウザのlocalStorageに保存されます。
- 複数端末でのリアルタイム共有や本物の職員認証には、Firebase/Supabase等のサーバーが必要です。
- 通知はブラウザを開いた時に期限を確認する方式です。
- 写真を多く保存するとブラウザ容量を使います。
